Alert CSS

.alert-wrapper, .alert-details {
    overflow: hidden;
    transition: height 1000ms;
    height: 0px;
}

.alert-details.open .content {
    margin: 5px 0;
    padding: 5px 0;
    border-top: 1px #a94442 solid;
}
  • Here's where we see that transition: height; specifcation.
  • We're setting the duration to 1000ms so that it's more apparent. You would probably use something a little snappier in your application, like 600ms.
  • And you can see that initial height of zero.
  • And we set overflow: hidden; to make the content initially invisible.
